# W1D4 Assignment: The Vowel Counter

### The Mission
You need to write a program that analyzes a given word and calculates exactly how many vowels (a, e, i, o, u) it contains.

### Requirements
- Use a `for` loop to iterate through the `targetWord`.
- Use the `.length()` and `.charAt()` methods.
- Your logic must be **case-insensitive** (it must count both "E" and "e").
- **Do not** change the line that declares the `targetWord` variable. The auto-grader will inject different words to test your logic!

### Success Criteria
The auto-grader will test your code against multiple scenarios:
1. Standard mixed-case words ("Engineering")
2. Words with no vowels ("Rhythm")
3. Words that are entirely vowels ("Queueing")
